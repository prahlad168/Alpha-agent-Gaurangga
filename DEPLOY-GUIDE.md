---

## 🚨 UNTUK ENABLE GITHUB PAGES

### Step 1: Buat Repo PUBLIC (WAJIB)
```
1. Buka: https://github.com/prahlad168/Alpha-agent-Gaurangga/settings
2. Scroll ke BAWAH → "Danger Zone"
3. Klik "Change visibility"
4. Klik "Make public"
5. Konfirmasi dengan ketik nama repo
```

### Step 2: Enable GitHub Pages
```
1. Tetap di Settings
2. Klik "Pages" di sidebar kiri
3. Source: ✅ "Deploy from a branch"
4. Branch: ✅ "main" + ✅ "/ (root)"
5. Klik "Save"
```

### Step 3: Tunggu 2-5 menit ⏳

### Step 4: Buka! 🎉
```
https://prahlad168.github.io/Alpha-agent-Gaurangga/
```

## 📱 UNTUK HP ANDROID

1. Buka URL di atas di Chrome HP
2. Klik ⋮ (titik tiga kanan atas)
3. Pilih "Add to Home Screen" / "Tambahkan ke Layar Utama"
4. GAURANGA app siap digunakan! 🤖

```

```

