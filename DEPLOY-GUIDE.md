# 🚀 GAURANGA - Deployment Guide

## ⚠️ JIKA GITHUB PAGES TIDAK AKTIF

Jalankan script ini di terminal (HP atau komputer):

```bash
cd Alpha-agent-Gaurangga
chmod +x enable-pages.sh
./enable-pages.sh
```

### Atau Manual (2 menit):

**Step 1: Public Repo**
1. Buka: https://github.com/prahlad168/Alpha-agent-Gaurangga/settings
2. Scroll ke "Danger Zone"
3. Klik "Change visibility"
4. Pilih "Make public"

**Step 2: Enable Pages**
1. Settings → Pages
2. Source: Deploy from branch
3. Branch: **main**
4. Folder: **/ (root)**
5. Save

**Step 3: Tunggu 2-5 menit**

**Step 4: Buka**
```
https://prahlad168.github.io/Alpha-agent-Gaurangga/android-app/
```

---

## 📱 UNTUK HP ANDROID

1. Buka URL di atas di Chrome HP
2. Klik ⋮ (titik tiga)
3. "Add to Home Screen"
4. GAURANGA app siap!

## 🔧 JIKA MASIH GAGAL
