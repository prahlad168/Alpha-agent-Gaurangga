## 🔑 BACKUP & TRANSFER

### Cara Backup Aman:

1. Buka Alpha Gaurangga
2. Menu → Memory → Ekspor
3. Masukkan password (opsional tapi recommended)
4. File terenkripsi tersimpan (.gaurangga)

### Cara Restore:

1. Menu → Memory → Impor
2. Pilih file backup
3. Masukkan password
4. Data kembali seperti semula

### Perhatian:

- ✅ Backup HANYA di HP Pak Pur
- ✅ Tidak ada cloud backup otomatis
- ✅ Transfer ke HP lain pakai file terenkripsi
