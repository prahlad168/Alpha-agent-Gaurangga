## ⚙️ Configuration

```yaml
agent_id: daily-briefing-v1
trigger_commands:
  - laporan pagi
  - daily report
  - brief me
  - genesis status
output_format: markdown
schedule: daily_08_00
parent: gaurangga-alpha-v1
```
